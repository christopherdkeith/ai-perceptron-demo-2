Understanding the Perceptron:

In a nutshell: A perceptron is the simplest neural network—a single neuron that makes decisions based on its inputs. The key insight: the perceptron IS the boundary. In this 3D demo, four numbers (Weight_x, Weight_y, Weight_z, and Bias) define a plane that separates points into two groups. The neuron doesn't just use the plane—it is the plane, storing its definition directly in these four parameters.

Why three inputs? Real-world problems rarely involve just one or two features. Consider classifying whether a house is expensive: you'd look at square footage (x), number of bedrooms (y), and location score (z). Or diagnosing illness: temperature, heart rate, and blood pressure. Most real-world decisions depend on multiple factors working together. In fact, neurons with dozens, hundreds, or even thousands of inputs are common in modern AI. This 3D demo gives you a tangible visualization of how a neuron processes multiple inputs simultaneously—a principle that scales to any number of dimensions, though we can only visualize up to three!

This is how AI learns: The perceptron demonstrates the core principle behind all machine learning. We start with random values, check how many mistakes we're making, then adjust our parameters bit by bit to reduce errors. Learning isn't magic—it's systematic trial and error: measure mistakes, tweak the numbers, repeat until errors are minimized. Whether it's a simple perceptron with 3 inputs or a deep neural network with millions of parameters, the fundamental process remains the same: learning = finding the parameter values that best fit the data.

How it works:

Target Plane: z = Ax + By + C shown in red
Points above the plane are labeled as +1 (green), below as -1 (blue).

Multiple Inputs, One Decision: The perceptron combines three coordinates (x, y, z) into a single decision. Each coordinate represents a different feature of your data. The weights tell the neuron how important each feature is—a large weight means that feature strongly influences the decision. This is exactly how neurons work in real AI systems: they take many inputs, weight them by importance, sum them up, and make a binary decision.

Perceptron Learning: In each training round, the perceptron learns through these steps:

  1. Make predictions: For each point, the perceptron calculates (Weight_x × x-coordinate) + (Weight_y × y-coordinate) + (Weight_z × z-coordinate) + Bias. If the result ≥ 0, it predicts +1 (above), otherwise -1 (below).
  2. Find mistakes: Compare the prediction with the correct answer (whether the point should be +1 or -1 based on the target plane). If they don't match, the perceptron made an error.
  3. Adjust weights: When wrong, the perceptron updates its weights using:
    New Weight = Old Weight + (Learning Rate × Error × Input)
    This nudges the decision plane toward the correct orientation and position. Each input's weight adjusts independently based on that input's value.
  4. Repeat: The perceptron goes through all points, making corrections until it classifies everything correctly.

Decision Boundary: In 3D, the decision boundary is a plane. You can rotate the 3D view by dragging with your mouse to see how the perceptron positions the plane to separate the points!

Goal: Make the decision plane match the target plane by correctly classifying all points.

Getting Started:

  1. Click "Start Training" to watch the perceptron learn in 3D space.
  2. Watch the accuracy increase as the perceptron corrects its mistakes and learns the correct plane orientation.
  3. Rotate the view: Click and drag on the 3D chart to rotate and view from different angles. Use the "Reset Camera" button to return to the default view.
  4. Adjust the learning rate to see how it affects learning speed. Try 0.01 (slow and steady) vs 0.5 (fast but jumpy).
  5. Change the target plane coefficients (A, B, C) to create different classification problems.
  6. Generate new data points to see how the perceptron adapts to different point distributions in 3D space.
  7. Reset the perceptron to start learning from scratch with new random weights.
