Understanding the Perceptron:

In a nutshell: A perceptron is the simplest neural network, a network of just one neuron. The neuron consists of just three numbers: Weight_x, Weight_y, and Bias. In this scenario these three numbers define a line! By adjusting these numbers, the perceptron learns to position and angle a line that separates points into two groups. That's the key insight: the perceptron IS the line—it stores the line's definition in its weights and bias. Watch as these three numbers change during training to find the perfect line.

This is how AI learns: The perceptron demonstrates the core principle behind most artificial intelligence. We start with random values (or zeros), check how many mistakes we're making, then adjust our numbers bit by bit to reduce those errors. Learning isn't magic—it's systematic trial and error: measure mistakes, tweak the numbers, repeat until errors are minimized. From simple perceptrons to complex neural networks, this is the fundamental process: learning = finding the parameter values that best fit the data.

How it works:

Target Line: y = mx + b (or y = mx + c) shown in red
Points above the line are labeled as +1 (green), below as -1 (blue).

Perceptron Learning: In each training round, the perceptron learns through these steps:

  1. Make predictions: For each point, the perceptron calculates (Weight_x × x-coordinate) + (Weight_y × y-coordinate) + Bias. If the result ≥ 0, it predicts +1 (above), otherwise -1 (below).
  2. Find mistakes: Compare the prediction with the correct answer (whether the point should be +1 or -1 based on the red target line). If they don't match, the perceptron made an error.
  3. Adjust weights: When wrong, the perceptron updates its weights using:
    New Weight = Old Weight + (Learning Rate × Error × Input)
    This nudges the decision boundary toward the correct classification.
  4. Repeat: The perceptron goes through all points, making corrections until it classifies everything correctly.

Decision Boundary: The dashed orange line shows where the perceptron currently thinks the boundary is. Watch it move toward the red target line as training progresses!

Goal: Make the decision boundary match the target line by correctly classifying all points.

Getting Started:

  1. Click "Start Training" to watch the perceptron learn. The orange dashed line (decision boundary) will adjust as it learns.
  2. Watch the accuracy increase as the perceptron corrects its mistakes and the decision boundary moves toward the red target line.
  3. Test the perceptron: Once training has started, click anywhere on the chart to add a test point. The perceptron will predict whether it's +1 or -1, shown as a diamond marker.
  4. Adjust the learning rate to see how it affects learning speed. Try 0.01 (slow and steady) vs 0.5 (fast but jumpy).
  5. Change the target line slope and intercept to create different classification problems.
  6. Generate new data points to see how the perceptron adapts to different point distributions.
  7. Reset the perceptron to start learning from scratch with new random weights.
